1. APP CREATES GRAPHIC NOVELS. 
2. USER SPECIFIES NUMBER OF PANELS (BETWEEN 1 AND 50)
3. WORDS GO ABOVE IMAGES, NOT IN IMAGE-INTERNAL WORD-BUBBLES OR THOUGHT-BUBBLES. 
4. USER UPLOADS ESSAY. 
5. USER SPECIFICES WHAT KIND OF GRAPHIC NOVEL HE WANTS. HE COULD SAY "KEEP IT NON-FICTION"; "EXPOSITIONAL"; "THREE CHARACTERS (ONE SEXY WOMAN, A HANDSOME MAN, AND AN UGLY MAN)"...HE CAN SET ANY SPECIFICATIONS HE WANTS; AND THE APP WILL, WHILE FOLLOWING THOSE SPECIFICATIONS, TURN HIS WRITTEN INPUT INTO A GRAPHIC NOVEL OUTPUT. 
5.1 THE USER CAN ALSO UPLOAD A PHOTO OR AN IMAGE AND SAY "THIS IS WHAT THE MAN SHOULD LOOK LIKE"; BASICALLY, HE CAN UPLOAD CONTENT EITHER AS TEXT (PRIMARY) AS IMAGE (SECONDARY)
6. THE APP WILL USE VENICE AI API FOR PICTURE GENERATION AND (WHEN THE MATERIAL IS X-RATED OR CONTROVERSIAL) FOR THE TEXT-CREATION. 
7. WHEN TEXT IS STRICTLY EXPOSITIONAL (NOT RISQUE), APP WILL USE DEEPSEEK, CLAUDE, CHATGPT, OR VENICE. THE USER CAN CHOOSE FROM A DROPDOWN. 
8. DEEPSEEK, CLAUDE, CHATGPT, VENICE WILL BE REFERRED TO (IN APP) AS ZHI 1, ZHI 2, ZHI 3, ZHI 4
9. THE APP'S MAIN FUNCTION IS TO CNOVERT WRITTEN INPUT TO GRAPHIC NOVEL. 
10.INPUT CAN UPLOADED, TYPED, DRAGGED/DROPPED. 
11. INPUT CAN BE TXT OR DOC OR PDF. 
12. A SECOND FUNCTION OF THE APP IS TO CONVERT WRITTEN INPUT (E.G. AN ESSAY) INTO A THE TEXT (SCREENPLAY, EFFECTIVELY) FOR A GRAPHIC NOVEL. 
13. TO EXPLAIN: SUPPOSE USER UPLOADS SOME PHILOSOPHY PAPER OF HIS. WELL, HE CAN USE THIS SECOND FUNCTION TO CONVERT IT INTO A SCREEN PLAY FOR A VIABLE GRAPHIC NOVEL. 
14 IN FURTHERANCE OF THIS, THE USER CAN SPECIFY HOW HE WANTS THE CONVERSION TO HAPPEN. FOR EXAPMLE, IF HE UPLOADS A PHILOSOPHY PAPER, HE CAN SPECIFY THAT HE WANTS 
A NON-FICTION, EXPOSITORY GRAPHIC NOVEL. OR HE CAN SAY: 'KEEP THE SUBJECT MATTER THE SAME; BUT EMBED IT IN A STORY ABOUT A WOMAN CAUGHT BETWEEN TWO MEN...' 
15. A THIRD FUNCTION IS: USER UPLOADS AN IMAGE (E.G. A PHOTO); AND THE USER DESCRIBES HOW HE WANTS THIS IMAGE TO BE THE BASIS OF A GRAPHIC NOVEL. 
16. SO FOR EXAMPLE, USER UPLOADS A PHOTO OF A SERIOUS MAN AT HIS DESK; SAYS "USE THIS IMAGE AS THE BASIS FOR A GRAPHIC NOVEL ABOUT A MAN AND A WOMAN WHO BOTH HAVE THE SAME PSYCHOANALYST BUT DON'T KNOW IT..."
17. ALL ARTEFACTS CAN BE DOWNLOADED. 
18. GRAPHIC NOVELS: DOWNLOADABLE. 
19. SCREENPLAYS (IF GENERATED): DOWNLOADABLE. 
20. PRIMARY API KEY: VENICE AI (FOR IMAGES AND ALSO FOR X-RATED TEXT)
21. SECONDARY API KEYS: DEEPSEEK, CHATGPT. CLAUDE
